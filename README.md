# 游戏组队投票插件

一个为 AstrBot 设计的简单指令式插件，用于在群组中发起游戏组队。该插件完全不使用大语言模型（LLM），仅基于逻辑运行，轻量高效。

## 配置
在管理面板可以配置 `default_timeout`（单位：秒），默认为 300 秒。

## 使用方法
1. **发起投票**：发送 `/有没有人玩 [游戏名] [人数上限]`
   - 示例：`/有没有人玩 英雄联盟 5`
2. **加入投票**：发送 `/玩`
   - 机器人会提示当前人数进度。
3. **自动结算**：
   - 当人数达到上限时，机器人会自动点名并宣布组队成功。
   - 当时间到达预设的倒计时后，如果没有人满，机器人会列出当前已报名名单。
